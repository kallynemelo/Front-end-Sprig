# 🌾 Projeto de Desenvolvimento – Sprig (React + Vite + TypeScript)

## 📖 Descrição Geral
Este repositório representa o **front-end do sistema Sprig**, desenvolvido com **React + Vite + TypeScript**.

O Sprig é uma plataforma web criada para aprimorar o processo de **controle de estoque, logística e distribuição de sementes**, oferecendo uma interface moderna, responsiva e intuitiva, destinada a gestores, técnicos e usuários do programa.

---

## 🎯 O que foi desenvolvido no Coding (Front-end)

### ✔️ Interface Completa
- Layout responsivo
- Componentes reutilizáveis (tabelas, botões, formulários, cards)
- Navegação organizada por rotas
- Layout com sidebar, header e área principal de conteúdo

### ✔️ Páginas Criadas
- Login  
- Dashboard  
- Estoque  
- Lotes de Sementes  
- Entregas  
- Logística  
- Relatórios  
- Usuários  
- (Opcional) Configurações  

### ✔️ Funcionalidades Implementadas
- Rotas com React Router
- Tabelas dinâmicas com ações
- Formulários com validação
- Hooks personalizados
- Componentização avançada
- Consumo de API (quando conectada)

---

- **React 18**
